# Smart Chess Roadmap

This document outlines upcoming feature implementations and architecture milestones:

## Phase 1 (v1.0.0 - Patzer): Core Restructure & Styling (Current)
- [x] Flat modernized dark styling with borderless text frames.
- [x] Relocate files to standard `src/` template.
- [x] Interactive best-move suggestions using Stockfish integration.
- [x] Custom rematch modals for game over terminations.

## Phase 2 (v2.0.0 - Gambit): Game Move Analysis & Graph (Next)
- [ ] Post-match analysis graph displaying evaluation bar history.
- [ ] Classification of moves (e.g. Best Move, Blunder, Mistake).

## Phase 3 (v3.0.0 - Fianchetto): Sound Themes & Custom Profiles
- [ ] Retro 8-bit, Classic Wood, and futuristic audio theme packs.
- [ ] Custom profile selection with glassmorphic avatars shown in clocks.

## Phase 4 (v4.0.0 - Zugzwang): Achievements & Trophy Cabinet
- [ ] Unlockable achievement milestones (e.g. Scholar's Mate, Invincible).
- [ ] PGN database file imports (`data/pgn/`) and search indexing.
