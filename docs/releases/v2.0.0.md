> *Codename: Gambit*

## ЁЯУЦ Overview

Gambit represents the second milestone in the evolution of Smart Chess, extending the desktop client from a gameplay platform into an active learning workspace. This release introduces a post-match evaluation graph and real-time move classification markers, enabling users to analyze their tactical strengths and weaknesses directly within the application.

While the previous release (Patzer) focused on core engine integration and network capabilities, Gambit emphasizes local game analysis, move-by-move telemetry, and visual feedback systems to help players study past matches.

---

## ЁЯОп Objectives

This milestone was developed with the following goals:

* Introduce a visual post-match evaluation graph showing advantage swings over time.
* Implement move quality classifications (`Best Move`, `Inaccuracy`, `Mistake`, `Blunder`).
* Build interactive timeline navigation to review specific board positions.
* Improve post-game review performance using background Stockfish analysis threads.

* **Interactive Timeline Nodes**: Clicking move steps dynamically updates board views to inspect critical turning points (`ReplayViewerModal`).
* **Visual Thresholds**: Highlights significant advantage swings and tactical momentum shifts.
